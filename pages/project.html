ArgosGaze

Sistema para monitorar o impacto de um anúncio em um outdoors. Foi criado um algoritmo que simula câmeras capturando o rosto de pessoas que estão olhando por um certo tempo para o outdoor e assim vendo o anúncio. Com base nestes dados iremos saber quais anúncios e onde eles estão tendo maior impacto: bairros, centros, enfim, locais onde os anúncios poderiam ser avaliados. Esse projeto foi desenvolvido para apresentar a ideia para empresas de marketing.
